Prompt (FR) — Générer un site Next.js complet pour COMATRA FISH MARINE

Objectif : Générer un site web professionnel, responsive, accessible et prêt à déployer (Next.js) pour COMATRA FISH MARINE. Respecter exactement la palette de couleurs “Ocean” fournie, utiliser Montserrat pour les titres et Roboto pour le corps du texte, employer la bibliothèque react-icons pour les icônes et intégrer Google Maps pour les agences.

Palette de couleurs (utiliser ces hex)

#01356d (bleu très foncé) — couleur principale (headings / footer)

#016ca5 (bleu foncé) — accents / boutons secondaires

#0396c7 (bleu moyen) — boutons primaires / liens

#04bbdf (bleu clair vif) — hover / highlights

#90e0ef (bleu pastel) — fonds de sections / cartes

#beedf4 (très clair) — arrière-plans et surfaces


Polices

Titres : Montserrat (var. 600–800)

Texte corps : Roboto (var. 400–500) → Charger via Google Fonts dans _document ou next/head.


Framework / techno

Next.js (app dir ou pages selon préférence — générer en app/ si possible)

CSS : Tailwind CSS (configurer la palette ci-dessus dans tailwind.config.js) ou CSS Modules si l’utilisateur préfère.

Icônes : react-icons (ex : FiPhone, FiMapPin, FiMail, FaFacebook, FaInstagram, FaTwitter)

Intégration Google Maps interactive pour la page Agences.

Formulaire contact : backendless (Next.js API route) qui envoie email via SMTP (expliquer dans README comment configurer).

SEO : meta tags, Open Graph, Twitter Card, sitemap.xml et robots.txt.

Accessibilité : aria labels, focus states visibles, textes alternatifs pour images.
